DataSync

Dropbox for your data powered by PubNub the worlds largest global Real-time Network. DataSync is the ability to have apps always be up-to-date regardless of network state. Upon reconnect, all data is re-synced with the client app. Your data is replicated to all data centers and encrypted.

Real-time Network

Highly Available Global Real-time Network is mandatory for distributed consumers in order to provide a fast and reliable user experience. You can only get that fast "Snappy" feel with a Global Real-time Network.

Maximum Security

Security is a complex and widely growing requirement especially for Real-time Mobile and Web Messaging Systems like PubNub. SSL TLS Sockets, SHA256 Message Signing, ACL (Access Control Level) and AES256 Symmetric Crypto are industry standard methods for providing High Levels of security for your application.

PubNub provides a default baseline level of security, with optional High Security with industry standard solutions for Signaling and DataSync applications.
